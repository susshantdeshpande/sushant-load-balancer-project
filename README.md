# AWS Application Load Balancer Project

This project demonstrates how to deploy a web application using AWS EC2 instances and an Application Load Balancer.

## Architecture

User
↓
Application Load Balancer
↓
Target Group
↓
EC2 Instances (India Server / USA Server)

## Technologies Used

- AWS EC2
- AWS Application Load Balancer
- Target Groups
- Nginx Web Server
- HTML / CSS
- GitHub Pages

## Live Website

https://sushantdeshpande.github.io/sushant-load-balancer-project/

## Key Features

- Load balancing between multiple EC2 instances
- High availability architecture
- Static website hosted on GitHub Pages
- AWS infrastructure deployment
